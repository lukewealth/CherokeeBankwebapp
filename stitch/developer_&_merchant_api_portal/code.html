End-point

Create New Transfer

Initiate a gold-backed asset transfer across the Cherokee CNB global network. This endpoint supports internal $Chero transfers and external fiat-to-crypto bridging.

Post https://api.cherokeebank.com/v1/transfers

Request Parameters

Field Type Required Description
amount decimal Required The amount of $Chero or gold-equivalent to send.
recipient_id string Required Unique identifier for the destination vault.
currency enum Optional Default is CHERO. Supports XAU.
security

Secure Signing

All transfers must be signed using your RSA Private Key before submission to ensure transaction integrity.

schedule

Settlement Time

Cross-border SWIFT movements settle in T+1, while internal Chero-to-Chero transfers are near-instant.

Response Object

Returns a Transfer object if successful. A 403 Forbidden error is returned if your vault balance is insufficient.

Request Example 
curl --request POST \
  --url https://api.cherokeebank.com/v1/transfers \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "amount": "1250.00",
    "recipient_id": "vlt_987654321",
    "currency": "CHERO",
    "memo": "Payment for Q3 Hardware"
  }'
Success Response 
{
  "status": "success",
  "data": {
    "transfer_id": "trf_550e8400",
    "amount": "1250.00",
    "currency": "CHERO",
    "timestamp": "2023-11-24T14:22:01Z",
    "status": "pending_confirmation"
  }
}
